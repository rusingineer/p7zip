7-Zip Plugin

7-Zip is a plugin for manipulating archives. It makes all archive files look like usual folders in File Manager.

The most of operations with compressed files can be called using right-clicking on files in File Manager (or in Windows Explorer) and selecting appropriate command from menu.

Using 7-Zip

Opening archive

There are two ways to open archive file:

  • If you have appropriate file type associated with 7-Zip, then you can open file with 7-Zip by double-clicking the file or by right-clicking and selecting Open command.
  • You can right-click the archive file, point to 7-Zip, and then click the Open command item.
Extracting archive

There are two ways to extract files from archive:

  • To extract all files from archive right-click the archive file, point to 7-Zip, and then click the Extract files... command item.
  • To extract specified files from archive Open archive with 7-Zip, select items to extract and call Copy To... command.

Extract Dialog will appear.

Testing archive

To test archive right-click the archive file, point to 7-Zip, and then click the Test archive command item.

Creating and updating files archive

For creating or updating archive file right-click the file(s) or folder(s) you want to compress, point to 7-Zip, and then click the Add to archive... command item. Add to Archive Dialog will appear.
